📱 Xpenselator – Feature Showcase
1. Core: Hybrid Calculator & Expense Logger
Smart Input System: Functions as a standard calculator (Add, Subtract, Multiply, Divide) but allows users to instantly "tag" a result to a category (e.g., type 150, tap 🍔 Food).


One-Tap Categorization: 16+ preset categories with emoji indicators (Food, Rent, Travel, Fuel, Medical, etc.) plus a "Custom Item" option for unique entries .

Negative Entry Logic: The app intelligently prevents negative entries unless the specific category has a positive balance to offset it (prevents illogical data).


High Precision: Handles large figures (up to 1 Trillion) without UI breakage.

2. Advanced Bill Splitting (The "Bill Finalizer")
Global Split Mode: Allows splitting the Grand Total of a sheet among multiple people.


Fairness Algorithms: Includes an "Equal Split" button that divides the total perfectly, handling remainder pennies automatically .

Lockdown Mode: Once a bill is split and saved, the sheet enters "Locked Mode".

⛔ Input Freeze: Keypad and category buttons are disabled to prevent accidental edits to the agreed total.

🔓 Unlock Toggle: A dedicated button allows users to unlock the sheet if revisions are strictly necessary.


Split Management: Users can long-press the "--- BILL SPLIT ---" header in the summary panel to delete the entire split group while keeping the expense data intact .

3. Multi-Sheet Project Management
Multiple Expense Sheets: Users can manage distinct "Sheets" (e.g., "Goa Trip", "Home Renovation", "Monthly Budget").


Gesture Navigation: Swiftly switch between sheets by flinging up or down on the screen, or by long-pressing the = button for a visual sheet selector.

Sheet Management:


Rename: Long-press the project name to customize the sheet title.

Delete: Option to delete entire sheets when no longer needed.


Freemium Limits: Supports a "Free Limit" (3 sheets) and a "Pro Mode" (Unlimited).

4. Analytics & Reporting (Pro Features)
PDF Report Generator: Generates professional-grade PDF reports.


Multi-Sheet Selection: Users can select which sheets to include in a single PDF .


Visual Breakdown: The PDF includes itemized lists, total calculations, and a visual bar chart of spending distribution .


In-App Visual Analytics: A built-in "Visual Analytics" overlay draws a horizontal bar chart directly on-screen for quick spending reviews .

Time Machine (History): A dedicated "History" tab allows users to scroll back through every single transaction chronologically.

5. Utility Station (Built-in Tools)
A dedicated dashboard  containing essential travel/finance tools:


💱 Currency Converter: Converts foreign currency to INR (stores the last used exchange rate).

📏 Distance Lab: Converts between Kilometers, Miles, and Feet.

⚖️ Weight Converter: Converts between Kilograms and Pounds.

6. UI/UX & Customization
Dual Theme Engine:

Dark Mode (Cyberpunk): High-contrast neon text (Cyan/Green) on black backgrounds for low-light usage.


Light Mode (Professional): Clean white backgrounds with blue/black text for daylight visibility .


Haptic & Sonic Feedback: Toggleable vibration and sound effects for a tactile calculator feel (customizable in Settings) .


Dynamic Visibility: Text colors automatically adapt (White vs. Black) based on the active theme to ensure readability.

7. Security & Monetization
Device-Lock Security: The "Pro Mode" unlock uses a unique Hardware ID generation method. The unlock code is mathematically tied to the user's specific device, preventing code sharing .


Direct Sales Channel: Integrated link to a Telegram Bot for purchasing Pro codes directly
